# 🗓️ React Date Counter

A simple yet effective date counter application built during Jonas Schmedtmann's React course to master **State Management**.

## 🚀 Features
- **Step Control:** Adjust the increment/decrement interval.
- **Count Control:** Increase or decrease the total number of days based on the current step.
- **Dynamic Date:** Real-time calculation of the date (past, present, or future) using JavaScript's `Date` object.

## 🛠️ Technologies Used
- **React.js** (useState hook)
- **JavaScript** (Date manipulation)
- **CSS3** (Custom UI with a modern purple theme and dashed borders)
